Información general

Piura es la capital de la región de Piura, en el noroeste de Perú. Es conocida por sus edificios coloniales, como la Catedral de Piura, con su altar dorado ornamental. La catedral está frente a la Plaza de Armas, una plaza pequeña con árboles de tamarindo. La Casa Museo Almirante Miguel Grau conserva fotos y recuerdos del héroe de guerra del siglo XIX. La Iglesia de San Francisco es donde Piura declaró su independencia de España en 1821

Etiquetas para dar estilo y formatos de texto.

    A continuacion veremos algunas:
  • Etiqueta "b"
    • Sin importancia semantica, solo da estilo negrita al atexto.
  • Etiqueta "strong"
    • Tiene importancia semantica ya que nos permite subrayar texto importante.
  • Etiqueta "i"
    • Sin importancia semantica, solo permite cambiar el formato a Italic.
  • Etiqueta "em"
    • Es Italic con importancia semantica, permite hacer énfasis en una parte del texto.
  • Etiqueta "small"
    • Sin importancia semantica, permite agregar texto pequeño.
  • Etiqueta "mark"
    • Sin importancia semantica, permite marcar o resaltara texto
  • Etiqueta "del"
    • Indicar que hemos borrado un texto.
  • Etiqueta "ins"
    • Indica que hemos insertado un texto.
  • Etiqueta "sub"
    • Texto pequeño inferior.
  • Etiqueta "sup"
    • Texto pequeño superior.

    Etiquetas para citar textos

      A continuacion veremos algunas:
  • Etiqueta "blockquote"
    • Se utiliza para denotar que el texto contenido en ella es una cita o sección de texto tomada de otro origen o fuente

    Ejemplo:

    "La imaginación es más importante que el conocimiento. El conocimiento es limitado, mientras que la imaginación abarca el mundo entero."

    Albert Einstein
    En este caso, "blockquote" se utiliza para envolver una cita de Albert Einstein, con el contenido de la cita y el nombre del autor dentro de la etiqueta "footer".

  • Etiqueta "cite"
    • Se usa para marcar el título de una obra, como libros, canciones, artículos, etc.

    Ejemplo:

    El libro Cien años de soledad es una obra maestra de Gabriel García Márquez.

    En este caso, "cite" se usa para marcar el título del libro "Cien años de soledad" como una obra citada en el texto.

    Etiquetas para imagenes

    La etiqueta que utilizaremos para insertar imágenes en nuestro sitio web se llama . Es una etiqueta abierta, es decir que no necesita una etiqueta de cierre, y nos permite renderizar imágenes de distintos formatos en nuestra página.

    Recibe 3 atributos super importantes

      A continuacion los veremos:
  • Atributo "src"
    • Indica la ruta donde está ubicada la imagen que queremos mostrar.
  • Atributo "alt"
    • Especifica un texto alternativo a la imagen, generalmente por accesibilidad y cuando no cargan las imágenes este es el texto que se mostrará en su lugar.
  • Atributo "title"
    • Permite definir un título para la imagen que se mostrará cuando el curso se ubique por encima de ella. Ejemplo:

    "img src ="nombre de la imagen a subir(imagen.jpg)"

    "alt ="Descripción de la imagen""

    "title =" cuando el usuario coloca el cursor sobre la imagen, se mostrará un mensaje emergente con el texto "Haz clic para ampliar"
